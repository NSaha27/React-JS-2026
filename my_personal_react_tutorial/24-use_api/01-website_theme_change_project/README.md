# DARK LIGHT THEME WEBSITE

---

## Key Points:

- In this basic website there will be some text and a button on toggle of which we can change the theme of the website from light to dark or vice versa.

### Tools used:

- React State (to store the current theme of the webpage)
- React Context API (ThemeContext)
- A context-provider function (ThemeContextProvider)
- Instead of "useContext" hook we have used the latest "use" API to get the values of the context
- Tailwindcss, to style the webpage
