# SHOPPING CART PROJECT

---

This is a e-commerce like patform relating to add online courses to cart for future puchasing.
The concept of React's Context API has been used here to manage states of components globally.

## Keys points:-

1. Full registration, authentication and authorization systems have been included,
2. An admin can add, update, and delete courses in this platform after authenticating himself,
3. An user can add courses to his cart, can remove courses from there or can clear the whole cart by clicking on a specific clear button.
4. Every user can manage their respective accounts.
