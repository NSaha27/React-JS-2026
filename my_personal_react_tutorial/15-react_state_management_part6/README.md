- This project should have the following -

* A landing page that will have a form to check whether the voter is authenticated or not, if yes, it will activate the voting machine (here ballot-box) for 3 minutes. After 3 minutes the voting machine will automatically be deactivated.
* In the digital ballot-box, There will be a list of all the candidates with their profile picture, name, and party-logo. Beside each of the candidate there will be a button, on pressing of which, that candiate will be voted. After a successful vote, the vote for that voter will be registered and he/she will be able to see, for some time, the candiate to whom he/she has given the vote, and after it, the page will be redirected to the landing page and that part of voting will be completed.
* The invigilator will be able to see the total number of voters, the number of voters that have given their votes, and the remaining number of voters in the list.
* The invigilator will have the permission to control the voting, i.e. to activate or deactivate the voting machine, or to reapprove the voter for voting if any glich happens.
* There will be a button, if pressed, will calculate the number of votes done, the number of votes not done, the percentage of number of votes done out of total number of votes, which candidate receives how many number of votes, and who wins the vote, etc.
