## Simple User Authentication System (Project)

### Key points:

- This is a user authentication system with "Registration", "Login", and "User Home" pages,
- User will register himself at first, then he has to log in to the system, after a successful log in he will be redirected to his home page.

### Tools used:

- Custom Hook (useUserContext),
- Context API with useContext Hook, and a Context Provider (UserContextProvider),
- React Router with useNavigate and useLocation Hooks, etc.
